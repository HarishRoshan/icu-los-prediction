# ICU Length of Stay Prediction (Supplementary Materials)

This repository contains code and processed datasets supporting the MSc dissertation:
"Early ICU Length of Stay Prediction Using First 24-Hour MIMIC-III Data".

## Data Access

Raw MIMIC-III data are not included due to licensing restrictions.
Researchers must obtain access via PhysioNet:
https://physionet.org/content/mimiciii/

## Included Data

All CSV files provided are derived, non-identifiable datasets generated
from MIMIC-III and are safe for sharing.

## Reproducibility

Steps:

1. Obtain MIMIC-III access
2. Run the preprocessing notebook
3. Train models and reproduce results

## Environment

See requirements.txt

This repository contains the implementation for the ICIMTH 2026 paper using MIMIC-III:

"Equitable Early Prediction of ICU Length of Stay from First-Day Clinical Data", authors: Harish JEYARAJ, Emeka ABAKASANGA, Thais WEBBER, and Dan DAI. Aston University, School of Computer Science and Digital Technologies, March 2026.
